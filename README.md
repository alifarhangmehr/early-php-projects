# Early PHP Projects (2013-2014)

A personal archive of web projects built during my early career as a PHP developer. These were developed around 1392-1393 Shamsi (2013-2014) and represent my first professional and academic work.

## Projects

### ghif/
**Sales & Accounting Software (نرم‌افزار فروش و حسابداری قیف)**
A complete point-of-sale and accounting system built from scratch. Features include sales management, inventory, reporting, keygen/licensing, and multilingual support.

### ghif-encrypted/
**Encrypted Distribution Version of Ghif**
The obfuscated version prepared for commercial distribution to clients. Includes installer and user manual.

### parking-project/
**Smart Parking System**
A Google Maps-based parking spot finder. Displays real-time availability of parking positions on a map with color-coded markers (green=available, red=occupied).

### northwest-method/
**Northwest Corner Method Solver**
An implementation of the Northwest Corner Method for solving transportation problems in Operations Research. Uses PHP for matrix generation and JavaScript for the algorithm computation and visualization.

### tavanmehr/
**Rehabilitation Center Automation System (سیستم جامع اتوماسیون مرکز توانبخشی مهر)**
Full management system for a rehabilitation center, including patient records, employee management, and administrative tools.

### parsianadsir/
**E-Commerce Platform**
A product catalog and online store system with categories, brands, product management, banner ads, shopping features, and a complete admin panel.

### topic-pos/
**Store Search System (فروشگاه تاپیک)**
A point-of-sale product search interface with autocomplete, powered by the Ghif accounting backend.

### university-portal/
**University News Portal**
A content management portal built for an Islamic Azad University branch. Features news publishing, user management, comments, and admin panel.

### custom-cms/
**Custom CMS Framework**
A content management system with user registration/login, news articles with comments, admin panel, student bulletin board, and rich text editing (Xinha).

### teacher-platform/
**Educational Platform**
A course/teaching website with user registration, file downloads, audio player, student board, RSS feed, and contact system.

---

## Tech Stack
- **Backend:** PHP 5, MySQL
- **Frontend:** HTML/CSS, JavaScript, jQuery
- **Tools:** XAMPP, Dreamweaver
- **Era:** Pre-framework PHP (no Laravel, no Composer)

## Note
This is archival code from my early career. It uses patterns typical of the era (direct SQL queries, mixed HTML/PHP, table-based layouts). It does not represent my current development practices but documents my professional journey.
